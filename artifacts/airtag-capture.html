Plain english

What is a "preamble code"?

Every UWB frame begins with a preamble: a known burst of pulses the receiver listens for to realise "a frame is starting." The exact pulse pattern is selected by a small number — the preamble code (on channel 9, codes 9–12 are legal). The receiver only "hears" a transmitter using the same code — it is a lock-and-key handshake, like two walkie-talkies that must be on the same sub-channel to hear each other.

We did not know which code Apple uses, so we swept all four. Code 9 stayed dead silent even with the AirTag actively ranging — wrong key. Codes 10–12 lit up the instant we tuned to them. That sweep is how we found Apple's PHY, and it is the proof the signal is real and not stray interference.

Anatomy

What a ranging frame is made of — and what we can see

Apple uses an IEEE 802.15.4z "STS" frame. The parts that make the radio detectable travel in the clear; the parts that carry the secret distance measurement are encrypted. That is why we can prove the transmission exists but cannot read the range itself.

What actually happened

  1. 1The iPhone and AirTag opened a UWB ranging session on channel 9 to measure distance to a few centimetres. This is what powers the "Find Nearby" arrow.
  2. 2My receiver sat on the same channel and swept preamble codes until it matched theirs. Code 9 heard nothing; 10–12 lit up immediately.
  3. 3Every frame's arrival is visible — timing, signal, preamble — but the ranging core is sealed with Apple's STS encryption. You see the conversation happening; you cannot read the words.
